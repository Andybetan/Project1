# Project1

Desarrollo en HTML para una pagina coorporativa. Cuenta con 3 paginas:

1 Home
2 Detalle Proyecto
3 Contacto

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1 Home

En la Home podrás ver los enlaces en el Navbar que te llevan a las secciones mediante un ancla. Para hacer un desplazamiento suave he utilizado una función en JavaScript. 

Encima del footer también he añadido una sección donde puedes solicitar contacto mediante la introducción del mail. Una vez añadido muestra un mensaje en la parte superior especificando el envio correcto. 

2 Detalle Proyecto

Accesible mediante el Learn More de los proyectos en la página principal podemos ver un mayor detalle del proyecto de la empresa. 

Además, podemos ver funcionalidades como una función de javascrip que anima las imagenes de proyectos. 

3 Contacto

Una vez rellenado el formulario muestra un mensaje en la parte superior especificando el envio correcto. 

Además, he añadido controles del contenido de los campos del formulario para verificar que los datos sean correctos. Ej. formato correo.
